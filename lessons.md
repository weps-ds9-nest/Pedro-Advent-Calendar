# Day 2: Next.js Routing
**Description:** Learn the ins and outs of the Next.js App Router and dynamic paths.
**Tip:** Always use Server Components by default unless you need client-side interactivity.

Today we're building routes!
Next.js uses a file-system based router where folders define routes.

Example route structure:
```ts
src/app/
  page.tsx          // matches /
  lesson/
    [id]/
      page.tsx      // matches /lesson/1, /lesson/2, etc.
```

Let's explore routing further in the next steps.
